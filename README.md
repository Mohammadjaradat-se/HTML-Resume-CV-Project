# HTML Resume / CV Project

This project is a **static resume (CV) web page** built using **HTML only** as part of a **Bootcamp training assignment**.

## Project Overview
The project demonstrates how to structure a personal resume using basic HTML elements.  
It includes multiple sections such as profile information, education, experience, projects, skills, and languages.

## Technologies Used
- HTML5

## Purpose
- Practice structuring content using HTML
- Learn how to build a resume layout with semantic HTML elements
- Apply HTML fundamentals in a slightly more detailed project
- Reinforce understanding through a practical Bootcamp exercise

## Notes
- No CSS or JavaScript is used
- The layout is intentionally simple and text-based
- This project is for educational and practice purposes only
